Features of this mini project, Weather Website

Awesome features, You won't expect on a Weather Website
Authorization and Authentication
Yes hard to beleive but, It's a weather website with login and signup system. Also secured as it uses modern hashing alogorithm
CRUD Operation
Yes hard to beleive again but, It's a weather website with CRUD operation.Create, Read, Update And Delete. All included on this mini project
Add Favorites
You can add your favorite city as a widget and Easily access its weather with ease with just a click.It will be saved until you delete it and even if you reload the website.
Admin Page
This project also features a admin page. Admin can delete or freeze the user's account. Admin can also increase the max favorites count of the users.
Full Weather Breakdown
You won't lack any weather details.Hourly Forcast, 10 days Forecast, AQI, Humidity, Precipitation Percent, And much more.
Accurate Weather Data
All the weather data are collected from a trusted and accurate open source API from Open Meteo.
