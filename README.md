# Weather Web App

## Overview
A simple web application to display current weather conditions and forecasts.

## Features
- Real-time weather data
- Search by city
- Temperature and humidity display
- 5-day forecast
- Responsive design

## Installation
```bash
git clone <repository-url>
cd weather-web
npm install
```

## Usage
```bash
npm start
```
Open `http://localhost:3000` in your browser.

## Technologies
- HTML/CSS
- JavaScript
- Weather API (OpenWeatherMap)
